## 1. Tener python venv

```
python -m venv venv
source venv/bin/activate
```
Si ya lo tienes instalado, solamente
```
source venv/bin/activate
```

## 2. Instalar selenium

```
pip install -r requirements.txt
```

## 3. Correr los test
Por ahora solo se tiene el test de agenda online en el ambiente de pruebas new
